## TL;DR

- Lyria 3 Pro は **最大3分** のフルレングストラックを **48kHz ステレオ** で生成するAI音楽モデル
- モデルID は `lyria-3-pro-preview`（フル）と `lyria-3-clip-preview`（30秒）の2種類
- 料金は Pro が **$0.08/曲** 、Clip が **$0.04/曲** （[Gemini API Pricing](https://ai.google.dev/gemini-api/docs/pricing) より）

## API セットアップと基本実装

### 前提条件

- Python 3.10 以上
- Google AI Studio のAPIキー（[取得手順](https://ai.google.dev/gemini-api/docs/api-key)）
- `google-genai` パッケージ

```bash
pip install google-genai
```

### テキストプロンプトによる楽曲生成

最もシンプルな方法は、テキストプロンプトで楽曲のジャンル・ムード・楽器編成を指定する方法である。以下は [公式APIドキュメント](https://ai.google.dev/gemini-api/docs/music-generation) に基づいたPython実装例になる。

```python
from google import genai
from google.genai import types

client = genai.Client(api_key="YOUR_API_KEY")

# 30秒クリップの生成
response = client.models.generate_content(
    model="lyria-3-clip-preview",
    contents="Create a 30-second cheerful acoustic folk song with guitar and light percussion",
    config=types.GenerateContentConfig(
        response_modalities=["AUDIO", "TEXT"],
    ),
)

# 音声データの取得と保存
for part in response.candidates[0].content.parts:
    if part.inline_data and part.inline_data.mime_type.startswith("audio/"):
        with open("output.mp3", "wb") as f:
            f.write(part.inline_data.data)
        print("Audio saved: output.mp3")
    elif part.text:
        print(f"Generated lyrics:\n{part.text}")
```

`response_modalities` に `["AUDIO", "TEXT"]` を指定すると、生成された音声データと歌詞テキストの両方が返却される。音声のみが必要な場合は `["AUDIO"]` とする。

### フルレングストラック（Pro モデル）

Lyria 3 Pro でフルレングストラックを生成する場合は、モデルIDを `lyria-3-pro-preview` に変更する。

```python
response = client.models.generate_content(
    model="lyria-3-pro-preview",
    contents="A cinematic orchestral piece building from a quiet piano intro to a full symphony crescendo, 2 minutes",
    config=types.GenerateContentConfig(
        response_modalities=["AUDIO", "TEXT"],
    ),
)
```

Pro モデルでは出力形式として MP3 と WAV の両方がサポートされている。48kHz ステレオのプロフェッショナルグレード音声が生成される。

## 高度な生成制御

### カスタム歌詞とセクション構造

[公式ドキュメント](https://ai.google.dev/gemini-api/docs/music-generation) によると、`[Verse]`、`[Chorus]`、`[Bridge]` といった構造タグを歌詞に埋め込むことで、楽曲の構成を細かく制御できる。

```python
lyrics_prompt = """
Genre: indie pop, upbeat tempo, acoustic guitar and synth

[Intro]
(Instrumental, light acoustic guitar picking)

[Verse 1]
Walking through the morning light
Every step feels just right
Code is running, tests are green
Best deployment I've seen

[Chorus]
Ship it, ship it, let it fly
Push to main and wave goodbye
Ship it, ship it, to the sky
Version two is standing by

[Bridge]
(Synth solo, building energy)

[Outro]
(Fade out with acoustic guitar)
"""

response = client.models.generate_content(
    model="lyria-3-pro-preview",
    contents=lyrics_prompt,
    config=types.GenerateContentConfig(
        response_modalities=["AUDIO"],
    ),
)
```
