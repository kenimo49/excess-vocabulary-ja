# 【物流AI #20】CO₂制約付き配送最適化：グリーン物流の実装

## なぜ CO₂ 制約が必要か

ESG 規制と顧客からのサステナビリティ要求が高まり、配送コストだけでなく**炭素排出量**も最適化目標に加える必要が出てきた。多目的最適化で「コスト vs CO₂」のトレードオフ解集合（パレートフロント）を求める。

## 車両排出量モデル

```python
import numpy as np

def co2_kg(dist_km: float, load_kg: float,
           truck_type="diesel_4t") -> float:
    """COPERT モデルの簡略版"""
    params = {
        "diesel_4t":   {"base": 0.25, "load_f": 0.0005},
        "diesel_10t":  {"base": 0.40, "load_f": 0.0003},
        "ev_4t":       {"base": 0.08, "load_f": 0.0002},
    }
    p = params[truck_type]
    return dist_km * (p["base"] + p["load_f"] * load_kg)
```

## 多目的 VRP の NSGA-II 実装

```python
import numpy as np
from pymoo.algorithms.moo.nsga2 import NSGA2
from pymoo.core.problem import Problem
from pymoo.optimize import minimize

class GreenVRP(Problem):
    def __init__(self, dist, demands, cap):
        n = len(demands)
        super().__init__(n_var=n, n_obj=2,
                         n_ieq_constr=1,
                         xl=0, xu=n-1, vtype=int)
        self.dist, self.demands, self.cap = dist, demands, cap

    def _evaluate(self, X, out, *args, **kw):
        costs, co2s, violations = [], [], []
        for route in X:
            c  = sum(self.dist[route[i], route[i+1]]
                     for i in range(len(route)-1))
            e  = co2_kg(c, sum(self.demands[route]))
            vc = max(0, sum(self.demands[route]) - self.cap)
            costs.append(c); co2s.append(e)
            violations.append(vc)
        out["F"] = np.column_stack([costs, co2s])
        out["G"] = np.array(violations).reshape(-1,1)

dist    = np.random.rand(10,10) * 50
demands = np.random.randint(10, 30, 10)
res = minimize(GreenVRP(dist, demands, cap=150),
               NSGA2(pop_size=100),
               termination=("n_gen", 200),
               verbose=False)
```

## パレートフロントの可視化と選択

```python
import matplotlib.pyplot as plt

F = res.F
plt.scatter(F[:,0], F[:,1], c="steelblue")
plt.xlabel("配送コスト（円）")
plt.ylabel("CO₂排出量（kg）")
plt.title("コスト vs CO₂ パレートフロント")
plt.show()

# CO₂ を 20% 削減しつつコスト最小の解を選択
mask    = F[:,1] <= F[:,1].min() * 1.2
best_i  = np.argmin(F[mask, 0])
best    = res.X[np.where(mask)[0][best_i]]
```

## 実運用の組み合わせ

| 手法 | 特徴 |
|---|---|
| NSGA-II | パレートフロント全体を取得 |
| 重み付きスカラー化 | 単一解が欲しい場合 |
| RL（多目的報酬） | リアルタイム動的ルーティング |

EV トラックと混合車両の場合は `truck_type` パラメータをルート割当て変数に加えることで、**車両タイプの選択**も同時最適化できる。
