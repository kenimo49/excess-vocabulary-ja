
SQLのINNER JOINとOUTER JOINの違いをわかりやすく、ここに記す。
結論、違いは「結合条件に一致しないデータをどう扱うか」にある。

### 元データ

#### users（ユーザー）
| id | name |
|----|------|
| 1  | 田中 |
| 2  | 鈴木 |
| 3  | 佐藤 |

#### orders（注文）
| id | user_id | item |
|----|---------|------|
| 1  | 1       | 本   |
| 2  | 1       | ペン |
| 3  | 2       | ノート |
| 4  | 4       | かばん |

特徴
- user_id=4はusersに存在しない
- 佐藤は何も注文していない

---

### INNER JOIN（内部結合）

#### 特徴
- 両方に存在するデータのみ取得
- 不一致データは除外

#### SQL
```sql
SELECT *
FROM users u
INNER JOIN orders o
ON u.id = o.user_id;
````

#### 結果

| user_id | name | order_id | item |
| ------- | ---- | -------- | ---- |
| 1       | 田中   | 1        | 本    |
| 1       | 田中   | 2        | ペン   |
| 2       | 鈴木   | 3        | ノート  |

---

### LEFT OUTER JOIN（左外部結合）

#### 特徴

* 左テーブルはすべて取得
* 一致しない場合はNULL

#### SQL

```sql
SELECT *
FROM users u
LEFT JOIN orders o
ON u.id = o.user_id;
```

#### 結果

| user_id | name | order_id | item |
| ------- | ---- | -------- | ---- |
| 1       | 田中   | 1        | 本    |
| 1       | 田中   | 2        | ペン   |
| 2       | 鈴木   | 3        | ノート  |
| 3       | 佐藤   | NULL     | NULL |

---

### RIGHT OUTER JOIN（右外部結合）

#### 特徴

* 右テーブルはすべて取得
* 一致しない場合はNULL

#### SQL

```sql
SELECT *
FROM users u
RIGHT JOIN orders o
ON u.id = o.user_id;
```

#### 結果

| user_id | name | order_id | item |
| ------- | ---- | -------- | ---- |
| 1       | 田中   | 1        | 本    |
| 1       | 田中   | 2        | ペン   |
| 2       | 鈴木   | 3        | ノート  |
| NULL    | NULL | 4        | かばん  |

---

### FULL OUTER JOIN（完全外部結合）

#### 特徴

* 両方すべて取得
* 不一致はNULL

### SQL

```sql
SELECT *
FROM users u
FULL JOIN orders o
ON u.id = o.user_id;
```

### 結果

| user_id | name | order_id | item |
| ------- | ---- | -------- | ---- |
| 1       | 田中   | 1        | 本    |
| 1       | 田中   | 2        | ペン   |
| 2       | 鈴木   | 3        | ノート  |
| 3       | 佐藤   | NULL     | NULL |
| NULL    | NULL | 4        | かばん  |

---

### 比較まとめ

| JOIN種類     | 不一致データの扱い |
| ---------- | --------- |
| INNER JOIN | 出ない       |
| LEFT JOIN  | 左は出る      |
| RIGHT JOIN | 右は出る      |
| FULL JOIN  | 両方出る      |
