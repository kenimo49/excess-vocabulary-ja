Microsoft Excelは、データ分析・可視化・管理のための多彩な機能を備えた、広く利用されている表計算ソフトです。一方、ODS（OpenDocument Spreadsheet）はオープン標準のスプレッドシート形式であり、LibreOfficeやApache OpenOfficeなど、さまざまなソフトウェアで閲覧・編集することができます。

これら2つの形式間での変換は、互換性の確保やファイル共有、あるいは特定の機能要件に対応するために必要となる場合があります。本記事では、Spire.XLS for .NETを使用し、C#でExcelとODSの相互変換を行う方法について解説します。

## Spire.XLS for .NET のインストール

まずは、Spire.XLS for .NET パッケージに含まれるDLLファイルを、.NETプロジェクトに参照として追加する必要があります。DLLファイルは、公式サイトからダウンロードするか、NuGet経由でインストールすることができます。、

```C#
PM> Install-Package Spire.XLS
```

## C#でExcelをODS（OpenDocument Spreadsheet）に変換する方法

Excel（.xls または .xlsx）ファイルは、Workbook.SaveToFile(string fileName, FileFormat.ODS) メソッドを使用することで、ODS形式として保存できます。基本的な手順は以下の通りです。

1. Workbook のインスタンスを作成する
1. Workbook.LoadFromFile() メソッドを使用してExcelファイルを読み込む
1. Workbook.SaveToFile(string fileName, FileFormat.ODS) メソッドを使用して、ODS形式で保存する

**サンプルコードは以下の通りです：**

```C#
using Spire.Xls;

namespace ConvertExcelToODS
{
    class Program
    {
        static void Main(string[] args)
        {
            // Workbookのインスタンスを作成
            Workbook workbook = new Workbook();

            // Excelファイルを読み込む
            workbook.LoadFromFile("Sample.xlsx");

            // ExcelファイルをODS形式で保存
            workbook.SaveToFile("ToODS.ods", FileFormat.ODS);
        }
    }
}
```

## C#でODSをExcel（XLS/XLSX）に変換する方法

Spire.XLS for .NETは、ODSファイルをExcel形式（XLSまたはXLSX）へ変換することもサポートしています。基本的な手順は以下の通りです。

1. Workbookのインスタンスを作成する
1. Workbook.LoadFromFile() メソッドを使用してODSファイルを読み込む
1. Workbook.SaveToFile(string fileName, FileFormat fileFormat) メソッドを使用して、XLSまたはXLSX形式で保存する

**サンプルコードは以下の通りです：**

```C#
using Spire.Xls;

namespace ODSToExcel
{
    class Program
    {
        static void Main(string[] args)
        {
            // Workbookのインスタンスを作成
            Workbook workbook = new Workbook();

            // ODSファイルを読み込む
            workbook.LoadFromFile("ToODS.ods");

            // ODSファイルをXLSX形式で保存
            workbook.SaveToFile("ToExcel.xlsx", FileFormat.Version2016);

            // ODSファイルをXLS形式で保存
            workbook.SaveToFile("ToExcel.xls", FileFormat.Version97to2003);
        }
    }
}
```
