## 4. 量子ニューラルネットワーク（QNN）による製造品質予測

### 4.1 変分量子回路をML層として使用

```python
import torch, torch.nn as nn, pennylane as qml, numpy as np
from sklearn.preprocessing import StandardScaler
from sklearn.metrics import classification_report, roc_auc_score

N_QUBITS = 8; N_LAYERS = 4
device = qml.device("default.qubit", wires=N_QUBITS)

@qml.qnode(device, interface="torch", diff_method="adjoint")
def quantum_circuit(inputs, weights):
    """変分量子回路（QNNコア）"""
    qml.AngleEmbedding(inputs * np.pi, wires=range(N_QUBITS), rotation='Y')
    qml.StronglyEntanglingLayers(weights, wires=range(N_QUBITS))
    return [qml.expval(qml.PauliZ(i)) for i in range(N_QUBITS)]

class QuantumLayer(nn.Module):
    """PyTorchと統合可能な量子回路層"""
    def __init__(self, n_qubits=N_QUBITS, n_layers=N_LAYERS):
        super().__init__()
        shape = qml.StronglyEntanglingLayers.shape(n_layers=n_layers, n_wires=n_qubits)
        self.weights = nn.Parameter(torch.Tensor(*shape).uniform_(-np.pi, np.pi))

    def forward(self, x):
        # バッチ逐次処理
        return torch.stack([torch.stack(quantum_circuit(s, self.weights)) for s in x])
```

### 4.2 ハイブリッドQNNアーキテクチャ

```
入力(32次元) → FC(64→32→8) → QuantumLayer(8qubits) → FC(8→16→2) → クラス確率
```

このアーキテクチャの利点：
- 古典層が高次元センサーデータを量子空間に適したサイズに圧縮
- 量子層が非線形・高次相関を捉える
- 後処理古典層が量子期待値を分類スコアに変換
- `diff_method="adjoint"` でシミュレーターが効率的な勾配を計算
