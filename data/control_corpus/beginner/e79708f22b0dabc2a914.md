```python
engine = QMLInferenceEngine()

@app.post("/predict/quality", response_model=QualityPrediction)
async def predict_quality(reading: SensorReading):
    """センサーデータから製品品質をQMLで予測"""
    try:
        result = await engine.predict(reading.sensor_values)
        logging.info(f"machine={reading.machine_id} status={result['status']} "
                     f"prob={result['defect_probability']:.3f} {result['processing_time_ms']:.1f}ms")
        return QualityPrediction(**result)
    except Exception as e:
        raise HTTPException(status_code=500, detail=str(e))

@app.get("/health")
async def health():
    return {"status":"healthy","requests":engine._count,"ts":datetime.now().isoformat()}

@app.get("/model/info")
async def info():
    return {"arch":"Hybrid QNN (8-qubit VQC)","n_qubits":8,"n_layers":4,
            "input":32,"classes":["正常品","不良品"],"backend":"PennyLane"}
```

**本番運用のポイント：**

- `asyncio.Lock` で量子回路の並行実行を排他制御（シミュレーターはスレッドアンセーフ）
- 実機量子クラウド（IBM Quantum等）利用時は非同期ジョブキューへ変更
- `/health` でヘルスチェック・Kubernetes Liveness Probe対応
- レイテンシ目標：シミュレーター < 200ms、実機量子クラウド < 5秒
