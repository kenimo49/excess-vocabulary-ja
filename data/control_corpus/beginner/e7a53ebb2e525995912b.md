

統計や機械学習の本を読んでいると、

- `p値が 0.05 未満なら有意`
- `95%信頼区間が 0 をまたがなければ有意`

のような説明が出てきます。

ここでよく起きる混乱が、

:::note
結局、p値と信頼区間って何が違うの？
:::

というものです。

この記事では、ダミーデータを使って、

- **p値は何を表しているのか**
- **95%信頼区間は何を表しているのか**
- **両者はどうつながっていて、何が違うのか**

を整理します。  
コードは **Google Colab でそのまま実行可能** です。

## 先に結論

いちばん大事な違いはこれです。

| 項目 | p値 | 95%信頼区間 |
| --- | --- | --- |
| 何を見るか | **ある1つの仮説**がデータとどれくらい矛盾するか | **もっともらしい母数の範囲** |
| 出力 | 1つの数 | 区間 |
| たとえば | 「母平均が70点だとすると、このデータはどれくらい珍しいか？」 | 「母平均は何点から何点くらいがありそうか？」 |
| 強み | 仮説検定に向く | 効果の大きさと不確かさが分かる |

ひとことで言うと、

:::note
**p値は「1つの値を疑う」道具**  
**信頼区間は「ありそうな値の範囲を見る」道具**
:::

です。

## 用語説明

今回の記事では、次の言葉が重要です。

- **標本（sample）**  
  手元で観測したデータ。今回は「12人の小テストの点数」です。
- **母平均（population mean）**  
  本当は知りたい平均値。たとえば「この教材を十分たくさんの学生に使ったときの、本当の平均点」です。
- **帰無仮説（null hypothesis）**  
  まず仮に置く基準の仮説。今回は  
  `H0: 母平均は 70 点である`
- **対立仮説（alternative hypothesis）**  
  帰無仮説に対抗する仮説。今回は  
  `H1: 母平均は 70 点ではない`

## 今回のダミーデータ

新しい教材を使って 12 人に小テストをしてみた、という設定にします。  
基準となる点数は **70点** とします。

```python
import numpy as np
import matplotlib.pyplot as plt
from scipy import stats

# 固定のダミーデータ
scores = np.array([72, 68, 75, 74, 69, 77, 73, 71, 70, 72, 74, 79])

# 比較したい基準値（帰無仮説の母平均）
mu0 = 70

n = len(scores)
df = n - 1
sample_mean = scores.mean()
sample_sd = scores.std(ddof=1)
se = sample_sd / np.sqrt(n)

# 1標本t検定（両側）
t_stat = (sample_mean - mu0) / se
p_value = 2 * (1 - stats.t.cdf(np.abs(t_stat), df=df))

# 95%信頼区間
t_crit = stats.t.ppf(0.975, df=df)
ci_low = sample_mean - t_crit * se
ci_high = sample_mean + t_crit * se

print(f"n = {n}")
print(f"sample mean = {sample_mean:.2f}")
print(f"sample sd = {sample_sd:.2f}")
print(f"t = {t_stat:.3f}")
print(f"two-sided p-value = {p_value:.3f}")
print(f"95% CI = [{ci_low:.2f}, {ci_high:.2f}]")
```

実行すると、次の値が出力されます。

```text
n = 12
sample mean = 72.83
sample sd = 3.21
t = 3.053
two-sided p-value = 0.011
95% CI = [70.79, 74.88]
```

この時点で見えていることは次の2つです。

- 標本平均は **72.83点**
- 95%信頼区間は **[70.79, 74.88]**

つまり、観測された12人のデータから見ると、母平均は **70.79〜74.88 点あたりがもっともらしい** と推定されます。

---

## p値とは何か

p値は、

:::note
**帰無仮説が正しいと仮定したとき、今のデータ以上に極端な結果が出る確率**
:::

です。

今回なら、帰無仮説は

$$
H_0: \mu = 70
$$

です。  
つまり

:::note
本当の母平均が 70 点だとしたら、今回みたいに標本平均が 72.83 点までズレることは、どれくらい珍しいか？
:::

を数で表したのが p値です。

今回の p値は

$$
p = 0.011
$$

でした。

これは、

:::note
**もし本当に母平均が70点なら、今回くらい極端なズレは 1.1% くらいしか起きない**
:::

という意味です。

### ここで重要

p値は、

- 「帰無仮説が正しい確率」
- 「この効果が本物である確率」

ではありません。

あくまで、

:::note
**帰無仮説を仮に正しいとしたときに、このデータがどれくらい珍しいか**
:::

です。

---

## 95%信頼区間とは何か

95%信頼区間は、

:::note
**このデータと整合的な母平均の範囲**
:::

と考えると、初心者には理解しやすいです。

今回の95%信頼区間は

$$
[70.79,\ 74.88]
$$

でした。

これは直感的には、

:::note
この12人のデータから見て、母平均としてありそうなのはだいたい 70.79 点〜74.88 点の範囲
:::

ということです。

### 厳密にはどういう意味？

厳密には、

:::note
同じ方法で標本を何度も取り、毎回95%信頼区間を作ると、そのうち約95%の区間が真の母平均を含む
:::

という意味です。

ただ、最初の理解としては、

:::note
**「母平均のもっともらしい範囲」**
:::

と捉えるとかなり分かりやすいです。

## 図1: 標本データ、標本平均、95%信頼区間を見る

まずは、実際の12人の点数と、標本平均・95%信頼区間を図にしてみます。

```python
x = np.arange(1, n + 1)

plt.figure(figsize=(8, 4.5))
plt.scatter(x, scores)
plt.plot(x, scores, alpha=0.5)

plt.axhline(mu0, linestyle="--", label=f"null mean = {mu0}")
plt.axhline(sample_mean, label=f"sample mean = {sample_mean:.2f}")
plt.fill_between(
    [0.5, n + 0.5],
    ci_low,
    ci_high,
    alpha=0.2,
    label=f"95% CI [{ci_low:.2f}, {ci_high:.2f}]"
)

plt.xlim(0.5, n + 0.5)
plt.xlabel("Student")
plt.ylabel("Score")
plt.title("Observed sample, sample mean, and 95% confidence interval")
plt.legend()
plt.grid(alpha=0.3)
plt.tight_layout()
plt.savefig("figure1_sample_ci.png", dpi=150, bbox_inches="tight")
plt.show()
```

## 図2: p値は「どれくらい珍しいか」を表す

次に、帰無仮説 `H0: μ = 70` のもとでの t分布を描いて、p値がどこに対応するかを見ます。

```python
x_t = np.linspace(-4.5, 4.5, 600)
y_t = stats.t.pdf(x_t, df=df)

plt.figure(figsize=(8, 4.5))
plt.plot(x_t, y_t, linewidth=2)

mask_left = x_t <= -abs(t_stat)
mask_right = x_t >= abs(t_stat)

plt.fill_between(x_t[mask_left], 0, y_t[mask_left], alpha=0.3)
plt.fill_between(
    x_t[mask_right], 0, y_t[mask_right],
    alpha=0.3, label=f"two-sided p-value = {p_value:.3f}"
)

plt.axvline(-abs(t_stat), linestyle="--")
plt.axvline(abs(t_stat), linestyle="--", label=f"observed |t| = {abs(t_stat):.2f}")

plt.xlabel("t value")
plt.ylabel("Density")
plt.title(f"t distribution under H0: mu = {mu0}")
plt.legend()
plt.grid(alpha=0.3)
plt.tight_layout()
plt.savefig("figure2_t_pvalue.png", dpi=150, bbox_inches="tight")
plt.show()
```

- これくらい極端な値が出る確率
