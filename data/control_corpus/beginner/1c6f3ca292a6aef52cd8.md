## 完成イメージ
以下のような動作をするモーダルを実装します。

**実装する機能:**
1. 「モーダルを開く」ボタンを押すとモーダルが表示される
2. 「閉じる」ボタンをクリックするとモーダルが非表示になる
3. 背景をクリックしてもモーダルが閉じる

## 実装手順
stimuluの導入、コントローラーの作成は省きます。

### 1. コントローラーの実装
```javascript:modal_controller.js
import { Controller } from "@hotwired/stimulus"

// Connects to data-controller="modal"
export default class extends Controller {
  static targets =["modal", "background"]

  open() {
    this.modalTarget.classList.remove("d-none")
    this.backgroundTarget.classList.remove("d-none")
  }

  close() {
    this.modalTarget.classList.add("d-none")
    this.backgroundTarget.classList.add("d-none")
  }

  closeOnBackground(e) {
    // 背景自体がクリックされた場合のみモーダルを閉じる
    // (モーダル本体のクリックでは閉じない)
    if (e.target == this.backgroundTarget) {
      this.close()
    }
  }
}
```
モーダルを開く、閉じるアクションと背景をクリックしたときも同様にモーダルが閉じるアクションを定義しています。

#### 解説

| コード | 説明 |
|:-|:-|
| `open()` |  ボタンがクリックされたときに実行されるメソッド |
| `this.modalTarget.classList.remove("d-none")` | モーダル本体を表示  |
| `this.backgroundTarget.classList.remove("d-none")` | 背景を表示 |
| `close()` | 閉じるボタンがクリックされたときに実行されるメソッド |
| `closeOnBackground(e)` | 背景クリック時に実行されるメソッド |
|`if (e.target == this.backgroundTarget){}`| クリックした要素が`this.backgroundTarget`と一致した場合に非表示 |

(補足)
- ```e.target```：
クリックされた要素。今回のケースでは背景(```this.backgroundTarget```)がクリックされた場合モーダル非表示になります！

### 2. Viewの実装
```erb:modal.html.erb
<div data-controller="modal" class="text-center" style="padding-top: 250px;">
  <!-- ボダン -->
  <button type="button" class="btn btn-primary" data-action="click->modal#open">
    モーダルを開く
  </button>

  <!-- 背景 -->
  <div data-modal-target="background" 
       data-action="click->modal#closeOnBackground"
       class="d-none position-fixed top-0 start-0 w-100 h-100 bg-dark bg-opacity-50"
       style="z-index: 99999;">
    <!-- モーダル本体 -->
    <div data-modal-target="modal" class="d-none position-fixed top-50 start-50 translate-middle bg-white p-3" style="z-index: 100000;">
      <h5>モーダルタイトル</h5>
      <p>ここにモーダルの内容が表示されます</p>

      <button type="button" class="btn btn-secondary" data-action="click->modal#close">
        閉じる
      </button>
    </div>
  </div>
</div>
```
```data-controller```,```data-modal-target```,```data-action```については前回の記事で説明していますのでそちらを確認してください！
