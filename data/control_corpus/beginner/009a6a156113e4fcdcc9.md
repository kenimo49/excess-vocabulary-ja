# 【物流AI #8】YOLOv8で棚の物品を認識：ピッキングミスをゼロに近づける

## 課題：ピッキングミスのコスト

誤品・数量違いは返品・再配送コストを生む。カメラ＋物体検出で「何を・何個取ったか」をリアルタイム検証する。

## YOLOv8 でカスタム学習

```python
from ultralytics import YOLO

# 事前学習済み nano モデルをベースに転移学習
model = YOLO("yolov8n.pt")
results = model.train(
    data="warehouse.yaml",   # クラス定義 + 画像パス
    epochs=50,
    imgsz=640,
    batch=16,
    device="cuda",
    augment=True,            # モザイク・フリップ・色変換
    patience=10              # 早期停止
)
```

`warehouse.yaml` の最小構成：
```yaml
path: ./dataset
train: images/train
val:   images/val
nc: 20
names: [item_A, item_B, ..., item_T]
```

## ピッキング検証ロジック

```python
def verify_pick(frame, order: dict,
                model, conf=0.5) -> dict:
    res   = model(frame, conf=conf)[0]
    boxes = res.boxes
    picked = {}
    for cls_id, conf_val in zip(
        boxes.cls.tolist(), boxes.conf.tolist()
    ):
        name = model.names[int(cls_id)]
        picked[name] = picked.get(name, 0) + 1

    errors = {}
    for item, qty in order.items():
        got = picked.get(item, 0)
        if got != qty:
            errors[item] = {"expected": qty, "got": got}
    return {"ok": len(errors)==0, "errors": errors}
```

## 精度向上のための実践Tips

- **データ拡張**: 照明変化・モーションブラーをシミュレート
- **ラベリング**: Roboflow でアノテーションを効率化
- **TTA（テスト時拡張）**: `augment=True` で推論精度+2〜5%
- **量子化**: TensorRT INT8 で推論速度 3〜4 倍

## 指標の目安

| 指標 | 目標 | 解説 |
|---|---|---|
| mAP@0.5 | ≥ 0.95 | 検出精度 |
| レイテンシ | ≤ 30ms | リアルタイム要件 |
| FP率 | ≤ 0.5% | 誤警報の抑制 |

精度向上には「難しいサンプル」を集中的に学習する **Hard Negative Mining** が有効。
