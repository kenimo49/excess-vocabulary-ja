今回は paiza の「グラフの s,t トレイル 3」の問題に挑戦！
<br>

# 🧩 問題概要

🔽 与えられるもの
・頂点数 `n` の無向グラフ（`1〜n`）
・始点 `s` と終点 `t`（s ≠ t）
・隣接リスト
<br>

🔽 グラフの特徴
・無向グラフ
・自己ループなし
・多重辺なし
<br>

🔽 やること
・`s` から `t` までのトレイルをすべて求める
<br>

🔽 トレイルの基本条件
・同じ辺は使えない
・頂点は何回通ってもOK
<br>

🔽 追加条件
👉 `s` と `t` を「通過点としても含む」必要がある
・`s` は最初だけじゃダメ → 途中でも出る必要あり（合計2回以上）
・`t` は最後だけじゃダメ → 途中でも出る必要あり（合計2回以上）
<br>

🔽 出力
・1行目：トレイルの総数
・2行目以降：各トレイル
　→ `s` で始まり `t` で終わる
　→ 途中にも `s` と `t` が最低1回ずつ出現

<br><br>

入力例:
```
4 2 3
3
2 3 4
3
1 3 4
3
1 2 4
3
1 2 3
```
出力例:
```
6
2 1 3 2 4 3
2 1 3 4 2 3
2 3 1 2 4 3
2 3 4 2 1 3
2 4 3 1 2 3
2 4 3 2 1 3
```

# ✅OK例：
```js
const rl = require('readline').createInterface({ input: process.stdin });

const lines = [];
rl.on('line', line => lines.push(line));

rl.on('close', () => {
    const [n, s, t] = lines[0].split(' ').map(Number);
    
    // 隣接リスト
    const graph = ['damy'];
    let idx = 1;
    for (let i = 0; i < n; i++) {
        const v = Number(lines[idx++]);
        graph.push(lines[idx++].split(' ').map(Number));
    }
    
    const visit = Array.from({ length: n+1 }, () => Array(n+1).fill(false)); // 隣接行列
    const trails = [];
    
    function dfs(v, trail, edges) {
        for (const next of graph[v]) {
            if (!edges[v][next]) {
                edges[v][next] = true;
                edges[next][v] = true;
                
                trail.push(next);
                
                if (next === t) {
                    const includesS = trail.filter(x => x === s).length;
                    const includesT = trail.filter(x => x === t).length;
                    if (includesS >= 2 && includesT >= 2) {
                        // s と t のどちらもトレイルに 2 回以上含まれる
                        trails.push([...trail]);
                    }
                    dfs(next, trail, edges); // t を通過して探索
                } else {
                    dfs(next, trail, edges);
                }
                
                edges[v][next] = false;
                edges[next][v] = false;
                
                trail.pop();
            }
        }
    }

    dfs(s, [s], visit);
    console.log(trails.length);
    trails.forEach(t => console.log(t.join(' ')));
});
```
<br>

## 🔍コードの流れ

🔽 全体の流れ
・入力を読み取る（`n, s, t` とグラフ）
・隣接リスト `graph` を作る
・「使った辺」を管理する配列 `visit` を用意
・結果を入れる `trails` を用意
・DFSを開始（始点 `s` から）
<br>

🔽 DFSの流れ
・現在の頂点 `v` からスタート
・隣接する頂点 `next` を1つずつ見る
<br>

🔽 探索条件
・まだ使っていない辺（`v → next`）のみ進める
<br>

🔽 次の頂点へ進むとき
・その辺を「使用済み」にする（両方向）
・`trail` に `next` を追加
<br>

🔽 分岐（ここが重要）
・`next` が `t` の場合
　→ `trail` 内で `s` と `t` の出現回数を数える
　→ 両方が2回以上なら結果に追加
　→ その後もDFSを続ける（ここがstep2との違い）
・`next` が `t` でない場合
　→ そのままDFSを続ける
 <br>
 
🔽 戻るとき（バックトラッキング）
・使った辺を「未使用」に戻す
・`trail` の最後の頂点を削除
<br>

🔽 最後
・すべての探索が終わったら
・トレイルの数を出力
・各経路を1行ずつ出力

<br><br><br><br><br>

# 📝まとめ

`s→t` の経路を全部試しながら、`s`と`t`を複数回通るものだけ記録するDFS
