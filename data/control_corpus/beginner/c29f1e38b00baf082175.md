# 【物流AI #9】AutoEncoderで設備故障を予兆検知する

## なぜ予兆検知か

コンベアやロボットアームが突然停止すると生産ラインが止まる。振動・温度・電流の時系列データを AutoEncoder で学習し、「正常パターンからの逸脱」を異常スコアとして検出する。

## LSTM-AutoEncoder

```python
import torch, torch.nn as nn

class LSTMAEcoder(nn.Module):
    def __init__(self, feat=4, h=32, seq=60):
        super().__init__()
        self.enc = nn.LSTM(feat, h, batch_first=True)
        self.dec = nn.LSTM(h,    h, batch_first=True)
        self.out = nn.Linear(h, feat)
        self.seq = seq

    def forward(self, x):           # x: (B, T, F)
        _, (hn, cn) = self.enc(x)
        # デコーダへ: 最後の隠れ状態を T 回繰り返す
        rep = hn.squeeze(0).unsqueeze(1).repeat(1, self.seq, 1)
        dec_out, _ = self.dec(rep)
        return self.out(dec_out)    # 再構成: (B, T, F)
```

## 正常データのみで学習

```python
model  = LSTMAEcoder()
opt    = torch.optim.Adam(model.parameters(), lr=1e-3)
loss_f = nn.MSELoss()

for ep in range(100):
    recon = model(X_normal)
    loss  = loss_f(recon, X_normal)
    opt.zero_grad(); loss.backward(); opt.step()
```

異常データを使わず「正常の再構成誤差を最小化」するだけ。異常時は誤差が急増する。

## 異常スコアと閾値

```python
def anomaly_score(model, x: torch.Tensor) -> float:
    with torch.no_grad():
        recon = model(x.unsqueeze(0))
    return ((recon - x)**2).mean().item()
```
