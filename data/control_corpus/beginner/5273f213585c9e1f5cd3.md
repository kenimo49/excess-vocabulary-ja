# 【物流AI #4】Qiskit で QAOA 回路を実装して VRP を解く

## QAOA 回路の構造

QAOA は **コストユニタリ** $U_C(\gamma)$ と **ミキサーユニタリ** $U_B(\beta)$ を交互に適用する。層数 $p$ を増やすほど精度が上がるがコストも増える。

```
|+>^n -> [U_C(g1) U_B(b1)] x p -> 測定
```

```python
from qiskit import QuantumCircuit
from qiskit.circuit import Parameter
import numpy as np

def qaoa_circuit(Q: np.ndarray, p: int = 2) -> QuantumCircuit:
    n = Q.shape[0]
    g = [Parameter(f"g{i}") for i in range(p)]
    b = [Parameter(f"b{i}") for i in range(p)]
    qc = QuantumCircuit(n)
    qc.h(range(n))
    for layer in range(p):
        for i in range(n):
            for j in range(i+1, n):
                if abs(Q[i, j]) > 1e-9:
                    qc.cx(i, j); qc.rz(g[layer]*Q[i,j], j); qc.cx(i, j)
            if abs(Q[i, i]) > 1e-9:
                qc.rz(g[layer]*Q[i,i], i)
        qc.rx(2*b[layer], range(n))
    qc.measure_all()
    return qc
```

## 古典最適化ループ（COBYLA）

```python
from qiskit_aer import AerSimulator
from scipy.optimize import minimize

def solve_qaoa(Q, p=2, shots=1024, n_restarts=3):
    sim = AerSimulator()
    qc  = qaoa_circuit(Q, p)
    best = (float("inf"), None)

    def cost(params):
        g, b = params[:p], params[p:]
        pd = {qc.parameters[i]: g[i] for i in range(p)}
        pd.update({qc.parameters[p+i]: b[i] for i in range(p)})
        counts = sim.run(qc.assign_parameters(pd),
                         shots=shots).result().get_counts()
        total = sum(
            cnt * (np.array([int(c) for c in reversed(bits)]) @ Q @
                   np.array([int(c) for c in reversed(bits)]))
            for bits, cnt in counts.items())
        return total / shots

    for _ in range(n_restarts):
        x0  = np.random.uniform(0, 2*np.pi, 2*p)
        res = minimize(cost, x0, method="COBYLA",
                       options={"maxiter": 200})
        if res.fun < best[0]: best = (res.fun, res.x)
    return best
```

## 実用上の注意

| 項目 | 説明 |
|---|---|
| 量子ビット数 | `nc x nv`（4顧客×2台 → 8量子ビット） |
| 推奨 `p` | 2〜3（増やすと精度向上・実行時間増） |
| 現在の限界 | 実機では〜20量子ビット程度が実用的 |
