### Git チーム運用ルール

 Git チーム運用ルールは、コミュニティが Git で共同作業する際に重要です。以下のルールを実践することで、Git チームの成果物が優れたものになるかもしれません。

#### 1. Git リポジトリの管理

 Git リポジトリは、すべてのコミットが変更したコンテンツを管理するものです。 Git チームは、リポジトリの管理を適切に実行します。

*   リポジトリのローカルワークとトリビューアワークを区別します。
*   リポジトリのフィールドを使用して変更を管理します。
*   リポジトリのバージョン管理を実行します。

*   コミットのメッセージは、コミットを実行する前に確認します。

#### 3. pull requestsのルール

 pull requestsのルールを実践することで、Git チームの成果物が優れたものになるかもしれません。
