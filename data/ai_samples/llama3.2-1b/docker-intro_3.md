# Docker入門ガイド

Dockerは、パフォーマンスを高めるためのオープンソースのビルデーションアプローチです。これは、コンポーネントを independent、self-sustaining、reproducibleの形で実装できるため、多くのエンジニアが Docker を学び、実践することをおすすめします。

## Dockerの基本概念

Dockerは、実行しているコンテナを、1 つのアプリケーションにコンテナ化して、モノラルで実行できるようにします。これは、コンテナを使用して、複数のアプリケーションを共存させることを可能にします。

*   **コンテナ**: Dockerコンテナは、実行しているアプリケーションに必要なすべてのアクセス権、ファイル、プロセスを包含します。
*   **イメージ**: Dockerイメージは、コンテナの基礎となるバージョンです。これは、イメージを使用して、コンポーネントを実行できるようにします。
*   **サンプル**: Dockerサンプルは、イメージを使用して、コンテナを実行できるようにします。

*   `docker-compose up` を使用して、コンテナ化します。
