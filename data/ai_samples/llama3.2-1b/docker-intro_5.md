# Docker入門ガイド
=====================================

 Docker は、Linux based UNIX環境上で、簡単に、多くのサービスを実装できる、パーソナライズドなコンテナのコンテナ化を可能にします。 Dockerは、Docker CLI、Dockerfile、Docker Swarmなどのツールを提供し、Docker の実践を容易にします。

### Docker Basics

 Docker は、OS上で実装される容器を実行できる、仮想化技術です。Container は、OS上に実装される実行状態の実体を表し、Docker は、Docker 守護、Docker セットアップなどの管理を実行する必要がありません。

### Docker Image

 Docker Image は、Docker 容器の基礎となる実体です。Image には、Docker によって実行されるさまざまなアプリケーションのコード、ファイル、ディレクトリー構造が含まれます。Image には、Docker Image 生成器 (Dockerfile) で生成される、コンポーネントのパッケージを指定する情報が含まれます。

### Docker Volumes

 Docker Volumes は、Docker Image 生成後、Docker 容器で使用する、ファイルとディレクトリーの構造です。Volumes は、Docker Image と Docker Image Generation の間でパーソナライズされ、Docker 容器で使用されるファイルとディレクトリーに影響を与えます。

### Docker Swarm

 Docker Swarm は、Docker Image 生成を管理するためのコンテナ管理プラットフォームです。Swarm には、Docker Images 1 つのアカウント上で共有できます。

### Docker CLI

 Docker CLI は、Docker Image 生成、容器の実行、Volumes の管理、およびother管理のためのコマンドです。

### Dockerfiles

 Dockerfiles は、Docker Image 生成に使用されるコンテナ化の基礎となる実体です。Dockerfiles には、Docker Image 生成で使用されるコード、ファイル、ディレクトリー構造を指定する情報が含まれます。

### Docker Swarm CLI

 Docker Swarm CLI は、Docker Swarm を実行するためのコマンドです。

### Docker Compose

 Docker Compose は、Docker Swarm を実行するためのコマンドです。

### Example Use Cases

*   **Docker Image Generation**: Docker Image Generation には、Docker Image 生成のためのコンテナ化の基礎となる実体を含めます。Dockerfile では、Docker Image Generation 1 つのアカウント上で共有できます。
*   **Containerization**: Docker Image Generation には、Docker Image Generation 1 つのアカウント上で共有できます。
*   **Docker Swarm**: Docker Swarm には、Docker Images 1 つのアカウント上で共有できます。
*   **Docker Compose**: Docker Swarm には、Docker Compose を実行するためのコマンドがあります。

### Conclusion

 Docker は、Docker Image Generation 1 つのアカウント上で共有できる、Docker 容器のコンテナ化を可能にする、パーソナライズドなコンテナのコンテナ化を可能にする、簡単に実行できる、OS上で実行される実体です。
