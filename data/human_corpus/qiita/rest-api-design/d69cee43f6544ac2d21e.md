
## APIとは？
APIとは、Application Programming Interfaceの略である。
APIの機能（役割）：システムとシステムをつなげる機能。
APIの基本構成：リクエスト&レスポンスでできている。
#### APIの基本動作：
他のシステムへ情報を依頼（リクエスト）
↓↑
他のシステムは情報を依頼主に返す（レスポンス）

## RESTとはREpresentational State Transferの略である
読み方は『レスト』である。

下記のルールに従って作成したAPIのことをREST APIということになる。
①アドレス可能性：URIではリソースの指定を行う。
②統一インターフェース：HTTPメソッド（POST、PUT、GET、DELETE）で追加、更新、取得、削除を設計する。
③ステートレス性：ステートレスであること（セッションといった状態管理をしない）(やり取りされる情報はそれ自体で完結して解釈することができる)
④接続性：HTML、XMLなどのハイパーメディアの使用すること（ハイパーリンクが使用できる）

## RestAPIメモ
### RestAPIUndelete(Apex) 
```Apex:(RestAPIUndelete)
@RestResource(urlMapping='/Undelete/*')
global with sharing class RestAPIUndelete {
    global class MyException extends Exception {}
    
    /** カスタムExceptionフラグ */
    global static Boolean isExceptionFlg = false;

    @HttpPost
    global static void getRestAPIUndelete() {
        
        RestRequest req = RestContext.request;
        RestResponse res = RestContext.response;

        // requestBody文字列から改行を取り除く
        String requestBodyStr = req.requestBody.ToString().replaceAll('\r\n|\n| ', '');
        // requestBody文字列から文字列【"id"】,【:】,【["】,【"]】,【{}】を取り除く
        requestBodyStr = requestBodyStr.replaceAll('"id"|\\:|\\["|"\\]|\\{|\\}', '');
        // requestBodyStr文字列【","】で分割する
        List<String> requestBodyStrList = requestBodyStr.split('","');

        String responseResult = '';
        List<ResultData> resultDataList = new List<ResultData>(); 

        try {
            if(isExceptionFlg) {
                throw new MyException();
            }
            // Undelete処理
            List<Database.UndeleteResult> UndeleteResultList = Database.undelete(requestBodyStrList,false);

            for(Database.UndeleteResult UndeleteResult : UndeleteResultList){
                // Undelete成功の場合
                if(UndeleteResult.isSuccess()){
                    ResultData resultData = new ResultData();
                    resultData.id = UndeleteResult.getId();
                    resultData.success = UndeleteResult.isSuccess();
                    resultDataList.add(resultData);
                }else {
                    ResultData resultData = new ResultData();
                    resultData.id = UndeleteResult.getId();
                    resultData.success = UndeleteResult.isSuccess();
                    resultData.errorMessage = UndeleteResult.getErrors().get(0).getMessage();
                    resultDataList.add(resultData);
                }
            }
            // APIレスポンス
            res.addHeader('Content-Type', 'application/json');
            res.statusCode = 200;
            responseResult = JSON.serialize(resultDataList);
            res.responseBody = Blob.valueOf(responseResult);
            return;
        } 
        catch (Exception e) {
            ResultData resultData = new ResultData();
            resultData.success = false;
            resultData.errorMessage = e.getMessage();
            res.addHeader('Content-Type', 'application/json');
            res.statusCode = 500;
            responseResult = JSON.serialize(resultData);
            res.responseBody = Blob.valueOf(responseResult);
            return;
        }
    }
    global class ResultData{
        global Id id;
        global Boolean success;
        global String errorMessage;
    }
}
```
