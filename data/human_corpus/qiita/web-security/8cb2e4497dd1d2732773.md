# はじめに
　令和5年春期に情報処理安全確保支援士の試験を受験して、1回の受験で合格できました。ちょうど1年前の今頃に勉強を始めていたので、これから受験する人の参考になればと思い、合格体験記を書きました。

# 筆者の職業、ステータス
　受験時は52歳のオッサンです。40歳台までは主に新システムの研究と開発に携わっており、特にセキュリティに特化した業務は担当していませんでした。50歳になって機密情報管理などの部門セキュリティ管理の仕事に携わることになり、支援士試験を受けてみようと思いました。普段の業務は社内手続きや社規を守った運用支援がメインで、支援士試験に頻出するWebアプリの脆弱性対策やアクセス制御・マルウェア対策などの業務は行っていないので、試験向けにしっかり勉強しました。
　また、一切免除なしで午前1～午後2まで受験するのは体力が持たないと思ったので、まずは令和4年秋期に応用情報技術者試験を受験して一発合格していました。

# 勉強したこと
### 午前対策
　参考書（徹底攻略　情報処理安全確保支援士教科書、インプレス）をひたすら読み込みました。試験の4か月前くらいから毎週末に勉強しました。午前2はとにかく参考書をしっかり覚えれば大丈夫です。最新のセキュリティ技術やマルウェア等を反映した問題も出るかと思ってTwitterで情報収集してましたが不要でした。60点以上で合格なので、参考書に載っている知識で解ける問題を取りこぼさなければ、まず落ちないと思います。

### 午後対策
　過去問をひたすら勉強しました（令和元年～直前分）。たぶんこれだけでも大丈夫とは思いますが、一応参考書として「情報処理安全確保支援士　午後問題徹底解説、技術評論社」も勉強しました。あまり色々な参考書に手を出すよりも、過去問をしっかりやって、判らない部分は調べて知識を補強する、ということを繰り返す方が効率が良いと思います。あとは本番に慣れるため、時間を測って、回答も指定文字数に納まるように練習しておくと試験当日も焦らずに済みます。一応「ポケットスタディ　情報処理安全確保支援士、秀和システム」も買いましたが、こちらはしっかり勉強するというより、気分転換に読むといった感じでした。

# 試験時のコツ
### 前日は良く寝て、体力温存しておく
　50歳台になるととにかく体力・集中力が落ちてます。長時間の試験なので、なるべく前日は良く寝て、少しでも体力を蓄積しておくことをおすすめします。私は試験中に疲れで目が霞んで試験問題がぼやけて見えるようになったので、ひたすら目薬を差して頑張りました。

### 昼食は食べない
　午後1の試験時間は90分、午後2は120分と長時間です。私は試験中にトイレに行きたくなったら困るので昼食は抜きました。

### 選択問題の時間配分を誤らない
　私は午後1の試験の途中で選択問題を変えてしまったので、時間が足りなくなりました。途中で変えると時間が足りなくなって焦るので、なるべくなら変えない方が良いです。（でも解いてる途中に自信がなくなって、他の問題に変えたくなるんですよね…）

### 記述式問題は、正解が判らなくても頑張って何か書く
　部分点があるかは判らないのですが、ずばりの正解が判らなくても、それらしい何かを頑張って書いた方が良いと思います。年齢が高ければ経験値も高いので、何かしらはきっと書けるはずです。

# 結果
　幸いにも1回で合格できました。しかし点数はイマイチで、午前2は70点台。午後1と午後2はどちらも60点台でした。ただ60点以上で合格なので、結果オーライです。

# おわりに
　受験は知力だけでなく、体力もある程度は必要なので、受けるならできれば若いうちの方が良いです。でも50歳台でも勉強すればちゃんと受かりますので、是非頑張って下さい。逆に50歳台で受けて良かった点は、試験当日の朝にちゃんと起きられることです！　またIPAがプレスリリースで合格者の年齢構成や最年長記録も発表しているので、自分がレアキャラである感が味わえます。
　受験される皆さま、合格目指して勉強がんばって下さい！
