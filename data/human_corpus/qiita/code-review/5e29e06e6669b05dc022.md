# 集計方法

## 集計期間：2023年1月1日～2023年12月29日

上記期間に投稿された記事でいいね数が30以上のものをまず抽出。抽出された1,600記事を、私のほうでタイトルや記事に付けられたタグを基に Python / データ分析関連の記事かをどうか選別し（タグがPython、機械学習、統計学、自然言語処理など）、選別された記事のいいね数の上位記事を今回掲載。なお、タグに「 Python 」と付いていても Python との関連が低いと思われる記事については今回の記事選定からは外しています。
